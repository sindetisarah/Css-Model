Css Box Model

A Css Box Model is essetially a Box That wraps around Every HTML Element

    It Consists of the Following:

  • Margin

    • Clears an area Outside the Border

  • Borders

    • It what goes around the padding and the Border

  • Padding

    • Clears an area around the content

  • Actual Content

    • The content of the Box
    It can either be a Text or an Image

Calculating the Total Dimension Of The Box

Finding the total is :

444px (width=400px)
+ 20px (left + right padding)=40px
+ 2px (left + right border)=4px
= 444px
